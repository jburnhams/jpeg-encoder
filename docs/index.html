About

This JPEG encoder is compiled from Rust to WebAssembly using wasm-bindgen. It demonstrates:

  • Real-time image encoding in the browser
  • Zero server-side processing - everything runs locally
  • Adjustable quality settings
  • Support for RGB, RGBA, and grayscale images
  • Streaming encoder for large images
